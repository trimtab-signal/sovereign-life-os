# Meshtastic

> Encrypted mesh comms on $28 LoRa radios — 2–15+ km range, no cell/internet. AES-256. Solar nodes run forever.

**Website:** [https://meshtastic.org](https://meshtastic.org)

**Replaces:** Garmin inReach ($150–400/yr sub)

## Installation

```
Flash firmware to supported LoRa hardware (T-Beam, T-Echo, Heltec V3, etc.)
Apps: https://meshtastic.org/docs/software/
Android, iOS, Web Client, Python CLI available.
```
