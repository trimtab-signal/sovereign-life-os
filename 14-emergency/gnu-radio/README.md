# GNU Radio + RTL-SDR

> $25 USB dongle → weather alerts, emergency broadcasts, radio scanner. Open-source signal processing.

**Website:** [https://gnuradio.org](https://gnuradio.org)

**Replaces:** Commercial scanner ($200–2,000)

## Installation

```
apt install gnuradio gqrx-sdr
RTL-SDR dongle: ~$25 from Amazon/AliExpress
GQRX for GUI: https://gqrx.dk/
Raspberry Pi compatible.
```
