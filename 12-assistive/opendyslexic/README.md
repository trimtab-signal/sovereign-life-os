# OpenDyslexic

> Free typeface reducing letter confusion for dyslexic readers. Bottom-heavy weighted letters. SIL-OFL license.

**Website:** [https://opendyslexic.org](https://opendyslexic.org)

**Replaces:** Dyslexie font ($70/yr)

## Installation

```
Download: https://opendyslexic.org/
Chrome extension: https://chrome.google.com/webstore/detail/opendyslexic/cdnapgfjopgaggbmfgbiinmmbdcglnam
Install system-wide or use CSS @font-face.
```
