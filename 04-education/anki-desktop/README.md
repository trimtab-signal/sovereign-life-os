# Anki Desktop

> Spaced-repetition flashcard desktop app. Free on all platforms except iOS.

**Website:** [https://apps.ankiweb.net](https://apps.ankiweb.net)

**Replaces:** Quizlet Plus ($36/yr)

## Installation

```
Download from https://apps.ankiweb.net
AnkiDroid (Android): Free on F-Droid
AnkiMobile (iOS): $24.99 on App Store
Point sync to your self-hosted anki-sync server.
```
